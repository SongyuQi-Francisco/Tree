# 🎄 Grand Luxury Christmas Tree - 在线版部署指南

## 📦 什么是在线版？

在线版是一个完全独立的 HTML 文件，可以直接部署到网上，无需本地服务器，任何人都可以通过链接访问。

## ✨ 在线版特点

- ✅ 单文件部署，无需配置
- ✅ 用户通过 UI 上传照片和音乐
- ✅ 使用浏览器 IndexedDB 存储数据
- ✅ 所有依赖使用 CDN（Three.js, MediaPipe）
- ✅ 支持手势交互、照片特写、背景音乐等全部功能
- ✅ 完全免费托管

## 🚀 快速部署

### 方法 1：GitHub Pages（推荐）⭐

**优点：**
- 完全免费
- 自动 HTTPS
- 稳定可靠
- 支持自定义域名

**步骤：**

1. **创建 GitHub 账号**（如果还没有）
   - 访问 https://github.com
   - 点击 Sign up 注册

2. **创建新仓库**
   - 点击右上角 + → New repository
   - 仓库名：`christmas-tree`（或任意名称）
   - 设置为 Public
   - 点击 Create repository

3. **上传文件**
   - 点击 uploading an existing file
   - 拖入 `index.html` 文件
   - 点击 Commit changes

4. **开启 GitHub Pages**
   - 进入 Settings → Pages
   - Source 选择 `main` 分支
   - 点击 Save

5. **访问网站**
   - 等待 1-2 分钟
   - 访问：`https://你的用户名.github.io/christmas-tree/index.html`

### 方法 2：Netlify Drop

**优点：**
- 最简单，拖拽上传
- 自动生成链接
- 免费 HTTPS

**步骤：**

1. 访问 https://app.netlify.com/drop
2. 将 `index.html` 拖拽到页面
3. 等待上传完成
4. 自动生成网站链接（如：`https://random-name.netlify.app`）

### 方法 3：Vercel

**优点：**
- 速度快
- 自动部署
- 免费 HTTPS

**步骤：**

1. 访问 https://vercel.com
2. 注册并登录
3. 点击 New Project
4. 上传 `index.html`
5. 点击 Deploy
6. 获得网站链接

## 🔧 常见问题

### Q: 为什么要有本地版和在线版？

- **本地版（test.html）**：适合自己使用，预加载照片和音乐
- **在线版（index.html）**：适合分享，用户自己上传内容

### Q: 在线版可以预设照片和音乐吗？

不推荐。因为：
1. 大量照片会让 HTML 文件巨大
2. 增加加载时间
3. 每个用户都会下载相同内容

如果需要预设内容，建议使用本地版。

### Q: 部署后可以更新吗？

可以！重新上传 `index.html` 即可：
- **GitHub Pages**: 提交新文件到仓库
- **Netlify**: 拖拽新文件到 Deploys 页面
- **Vercel**: 重新部署

### Q: 如何自定义默认文字？

编辑 `index.html`，找到第 111 行：

```html
<h1 id="display-line1" class="title-line">Merry Christmas</h1>
<h1 id="display-line2" class="title-line">2024</h1>
```

修改为你想要的文字，然后重新部署。

### Q: 在线版安全吗？

完全安全：
- 所有代码在浏览器本地运行
- 照片和音乐存储在用户自己的浏览器中
- 不会上传到任何服务器
- 不收集任何数据

## 📊 性能优化建议

1. **照片优化**
   - 建议每张照片 < 2MB
   - 推荐分辨率：1920x1080 或更小
   - 太大的照片会影响加载速度

2. **音乐优化**
   - 建议 < 10MB
   - MP3 格式，128-320kbps
   - 过大会影响存储和加载

3. **浏览器建议**
   - Chrome、Edge、Firefox 最佳
   - Safari 需要 iOS 14+ / macOS 11+
   - 建议使用最新版本
